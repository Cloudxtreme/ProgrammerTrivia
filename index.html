Name a common mistake made when using pointers in C.

  1. Using a pointer variable before it is allocated a space in heap (dynamic storage area). C will not recognize this as a syntax error.
    Example:

    int *p;
    *p = 5;
    /* A value of 5 has been set as the initial value of p. */

  2. Memory leak
  3. Dangling pointer

What is the name of the memory location allocated for items such as function calls durring recursion?

Heap

How do languages such as Pascal and Ada handle the process of reading to the end of a file.

Languages such as Pascal and Ada call a function that returns a boolean value to determine when the end of a file has been reached.

How are exceptions handled in C?

The C language does not use exception handlers. Instead, functions can return a value that indicates if an exception has occured.

Java supports only one parameter passing method, call-by-value. How then, is it possible that we are able to achieve the effect of call-by-reference?

We can achieve the effect of call-by-reference in Java by passing an object as a parameter. The object can contain attributes which can be modified to change the properties of the origional object.

A function written in C creates an array. The function then returns the array that was created. What is the result?

The result of returning an array from a function in C is that the local variable created in the function will be popped off the stack after the function call. This results in a returned address that points to an undefined location.

What is the difference between polymorphism and overloading?

Polymorphism is the ability for the body of a function with the same name as another to contain different data. Overloading is a simple form of polymorphism. Overloading allows a programmer to create functions with the same name, but different signatures.
